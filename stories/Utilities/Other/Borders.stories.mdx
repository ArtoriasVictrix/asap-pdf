# Borders

With these utility classes you can set:

- `border-style`
- `border-width`
- `border-radius`
- `border-color`

for every edge or corner of your elements.

## Border style

Set `border-style` of the current element with our `.border` utilities.

The format is `.border-{side}-{value}`.

Where *side* is one of:

- `t` - for classes that set `border-top-style`
- `b` - for classes that set `border-bottom-style`
- `l` - for classes that set `border-left-style`
- `r` - for classes that set `border-right-style`
- `x` - for classes that set both `border-left-style` and `border-right-style`
- `y` - for classes that set both `border-top-style` and `border-bottom-style`
- `blank` - for classes that set a `border-style` on all 4 sides of the element

Where *value* is one of:

- `none` - for classes that set `border-style` to `none`
- `hidden` - for classes that set `border-style` to `hidden`
- `dotted` - for classes that set `border-style` to `dotted`
- `dashed` - for classes that set `border-style` to `dashed`
- `solid` - for classes that set `border-style` to `solid`
- `double` - for classes that set `border-style` to `double`
- `groove` - for classes that set `border-style` to `groove`
- `ridge` - for classes that set `border-style` to `ridge`
- `inset` - for classes that set `border-style` to `inset`
- `outset` - for classes that set `border-style` to `outset`

## Border width

Adjust `border-width` of the current element with our `.border` utilities.

The format is `.border-{side}-{value}`.

Where *side* is one of:

- `t` - for classes that set `border-top-width`
- `b` - for classes that set `border-bottom-width`
- `l` - for classes that set `border-left-width`
- `r` - for classes that set `border-right-width`
- `x` - for classes that set both `border-left-width` and `border-right-width`
- `y` - for classes that set both `border-top-width` and `border-bottom-width`
- `blank` - for classes that set a `border-width` on all 4 sides of the element

Where *value* is in the range from **0** to **6** in 1 pixel increments.

## Border color

Set `border-color` of the current element with our `.border` utilities.

The format is `.border-{side}-{value}`.

Where *side* is one of:

- `t` - for classes that set `border-top-color`
- `b` - for classes that set `border-bottom-color`
- `l` - for classes that set `border-left-color`
- `r` - for classes that set `border-right-color`
- `x` - for classes that set both `border-left-color` and `border-right-color`
- `y` - for classes that set both `border-top-color` and `border-bottom-color`

- `warning` - for classes that set `border-color` to `var(--border-warning-cl)`
- `danger` - for classes that set `border-color` to `var(--border-danger-cl)`
- `info` - for classes that set `border-color` to `var(--border-info-cl)`
- `transparent` - for classes that set `border-color` to `transparent`

## Border radius

Adjust `border-radius` of the current element with our `.brs` utilities.

The format is `.brs-{corner}-{value}`.

Where *corner* is one of:

- `tr` - for classes that set `border-top-right-radius`
- `br` - for classes that set `border-bottom-right-radius`
- `bl` - for classes that set `border-bottom-left-radius`
- `tl` - for classes that set `border-top-left-radius`
- `blank` - for classes that set a `border-radius` on all 4 corners of the element

Where *value* is in the range from **0** to **32** in 4 pixel increments.

## No border

Remove borders of the current element with our `.no-border` utilities.

The format is `.no-border-{side}`.

Where *side* is one of:

- `t` - for classes that set `border-top`
- `b` - for classes that set `border-bottom`
- `l` - for classes that set `border-left`
- `r` - for classes that set `border-right`
- `x` - for classes that set both `border-left` and `border-right`
- `y` - for classes that set both `border-top` and `border-bottom`
- `blank` - for classes that remove a `border` on all 4 sides of the element
