# Float

These utility classes float an element to the left or right, or disable floating, using the
[CSS float property](https://developer.mozilla.org/en-US/docs/Web/CSS/float).

> Please be aware float utilities have no effect on flex items.

- .float-right
- .float-left
- .float-none

## Sass

### Utilities API

Float utilities are declared in the utilities API in `app/scss/_utilities.scss`.
[Learn how to use the utilities API on the Bootstrap website](https://getbootstrap.com/docs/5.1/utilities/api/#using-the-api).

```
'float': (
	property: float,
	values: (left, right, none),
),
```
