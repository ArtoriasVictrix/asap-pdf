# Checkbox

At first, it might seem like checkboxes make no sense in a framework used to create PDF files, but itâ€™s not true.
Typically, checkboxes are necessary to display that a user agrees to the terms and conditions of service.

ASAP PDF has two types of checkboxes: small ones (with `.checkbox--sm`
[modifier](https://en.bem.info/methodology/quick-start/#modifier)) and regular ones (without a modifier).
