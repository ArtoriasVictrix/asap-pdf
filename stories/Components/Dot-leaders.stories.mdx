# Dot leaders

A list of properties in which a property and its value are separated by dots is called a **dot leaders**. For such
lists, ASAP PDF has a `.param` component:

<Canvas>
<div class="w-300">
	<dl class="param">
		<dt class="param__prop">Your balance</dt>
		<dd class="param__value">$100</dd>
	</dl>
	<dl class="param">
		<dt class="param__prop">Expenses</dt>
		<dd class="param__value">$20</dd>
	</dl>
	<dl class="param">
		<dt class="param__prop">Received money</dt>
		<dd class="param__value">$120</dd>
	</dl>
</div>
</Canvas>
