# Horizontal Rule

The `.hr` class applied to `<hr>` element allows you to create horizontal dividers.

## Default

<Canvas>
<hr class="hr my-20" />
</Canvas>

## Light

The rule color can be changed to light gray by applying the `.hr--light`
[modifier](https://en.bem.info/methodology/quick-start/#modifier).

<Canvas>
<hr class="hr hr--light my-20" />
</Canvas>
