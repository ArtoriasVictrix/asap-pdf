# Header and footer of a document

In the ASAP PDF framework, [Puppeteer](https://pptr.dev/) is used to convert HTML to PDF. Puppeteer has a lot of
different [conversion options](https://github.com/puppeteer/puppeteer/blob/main/docs/api.md#pagepdfoptions) that include
header and footer settings.

ASAP PDF doesnâ€™t have default settings for header and footer. If you need to set the header and footer for a certain
document, next to its pug file you can create a JS file with the same name that will contain conversion settings. The
settings will be automatically applied at the compilation stage.
